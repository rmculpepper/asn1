3 DER Encoding and Decoding

procedure

(DER-encode type v) → bytes?

  type : asn1-type?
  v : any/c
Produces the DER-encoding of v as type.

procedure

(DER-decode type b) → any/c

  type : asn1-type?
  b : bytes?
Decodes the DER-encoding b of type to get a Racket value.

procedure

(DER-read type in) → any/c

  type : asn1-type?
  in : input-port?
Reads one DER encoding from in and decodes it as type.
